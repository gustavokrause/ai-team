# AI - Context & Memory

This folder holds the context, memory, and documentation of the AI professionals.

## Structure

### `professionals/`

**Individual context for each AI professional**

- Each professional keeps its own full context
- Specific competencies and responsibilities
- Personalized knowledge base

## Purpose

Keep the AI team's "institutional memory" organized and separate from code and product structures, making it easier to:

- **Preserved context**: each AI keeps continuity
- **Clear organization**: separation between AI context and product
- **Collaboration**: AI professionals can reference each other's contexts

## Usage

This folder is for:

- Context loading by the AIs
- Alignment between different AI professionals
- Keeping the system consistent
